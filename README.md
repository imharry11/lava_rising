一个Minecraft 1.21.11版本的岩浆上升生存插件<br>
/lavarise start 开始游戏<br>
/lavarise stop 强制停止游戏<br>
config.yml:<br>
  start-y: -64 #岩浆开始上升的坐标<br>
  stop-y: 319 #岩浆最终达到的坐标<br>
  rise-speed: 8 #岩浆多少tick上升一格<br>
  border-start-size: 1000 #边界在开始时的大小<br>
  border-change-size: 50 #边界最终的大小<br>
  border-time: 12000 #边界从开始缩到最终大小的时间，单位：tick<br>
  pvp-start-lava-y: 100 #岩浆上升到多少格开始玩家pvp<br>
